# tails-bootstrap

## Project Overview

A Bash script that:
- Builds a custom persistent-storage bundle for Tails OS
- Bundles PGP keys, OnionShare, and WireGuard configs
- Saves to a USB partition
- README doubles as a usage/tutorial guide
